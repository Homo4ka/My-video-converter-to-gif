# Video to GIF Converter

Простая и удобная CLI программа для конвертации видеофайлов в GIF форматы. Поддерживает как одиночную, так и пакетную конвертацию.

## Возможности

- Конвертация одного видео файла в GIF
- Пакетная конвертация нескольких видео
- Настройка FPS (кадров в секунду)
- Изменение размера выходного GIF
- Поддержка популярных видео форматов
- Интуитивно понятное меню навигации
- Отображение прогресса и информации о конвертации

## Поддерживаемые форматы видео

- MP4 (.mp4)
- AVI (.avi)
- MOV (.mov)
- MKV (.mkv)
- WebM (.webm)

## Требования

- Python 3.7 или выше
- FFmpeg (требуется для работы moviepy, который используется в программе)

### Установка FFmpeg

**Windows:**
1. Скачайте FFmpeg с [официального сайта](https://ffmpeg.org/download.html)
2. Распакуйте и добавьте путь к `bin` в переменную PATH
3. Или используйте Chocolatey: `choco install ffmpeg`

**macOS:**
```bash
brew install ffmpeg
```

**Linux (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install ffmpeg
```

**Linux (Fedora):**
```bash
sudo dnf install ffmpeg
```

## Установка и запуск

### 1. Клонирование и настройка

```bash
#клонируйте репозиторий
git clone <ваш-репозиторий>
cd video-to-gif-converter

#установите зависимости
pip install -r requirements.txt
```

### 2. requirements.txt
Создайте файл `requirements.txt` со следующим содержанием:
```
moviepy>=1.0.3
Pillow>=9.0.0
```

### 3. Запуск программы

```bash
python video_to_gif_converter.py
```

## Использование

### Главное меню
При запуске программы вы увидите главное меню:
```
==================================================
      КОНВЕРТЕР ВИДЕО В GIF
==================================================

ГЛАВНОЕ МЕНЮ:
1. Конвертировать одно видео
2. Конвертировать несколько видео
3. Справка и рекомендации
4. Выход
```

### Конвертация одного видео
1. Выберите пункт `1` в главном меню
2. Введите путь к видеофайлу
3. Укажите имя выходного файла (по умолчанию - имя видео с расширением .gif)
4. Выберите настройки конвертации:
   - Стандартные (FPS: 10, без изменения размера)
   - Высокое качество (FPS: 15)
   - Пользовательские настройки

### Пакетная конвертация
1. Выберите пункт `2` в главном меню
2. Выберите режим:
   - Конвертировать все видео в папке
   - Выбрать несколько файлов вручную
3. Укажите настройки конвертации для всех файлов
4. Укажите папку для сохранения результатов

### Справка и рекомендации
Пункт `3` содержит полезные рекомендации по настройкам:
- Оптимальные значения FPS
- Рекомендуемые размеры для социальных сетей
- Информация о поддерживаемых форматах

## Рекомендации по настройкам

### FPS (кадры в секунду)
| Качество | FPS | Размер файла | Применение |
|----------|-----|--------------|------------|
| Низкое   | 5-8 | Маленький    | Простые анимации |
| Среднее  | 10-12 | Средний     | Большинство видео |
| Высокое  | 15+ | Большой      | Важные моменты |

### Размеры
- **Стандартный**: 480x270 пикселей
- **Для соцсетей**: 640x360 пикселей
- **Высокое качество**: оригинальный размер

## Расширенные возможности

Для продвинутых пользователей доступны дополнительные настройки через изменение кода:

```python
#изменение качества сжатия (в файле video_to_gif_converter.py)
clip.write_gif(output_path, fps=fps, program='ffmpeg', 
               opt='OptimizePlus', fuzz=10)
```

## Устранение неполадок

### Ошибка "MoviePy Error: ..."
Убедитесь, что FFmpeg установлен правильно:
```bash
ffmpeg -version
```

### Большой размер GIF
- Уменьшите FPS до 5-10
- Уменьшите размер изображения
- Используйте более короткие видео

### Низкое качество
- Увеличьте FPS до 15-20
- Используйте оригинальный размер
- Проверьте качество исходного видео
